## MVC and Web Forms Project

**About Project**
- based on MVC and Web Forms software design patterns
- project consists of User (Web Forms) and Admin (MVC) site with CRUD operations on both
- login/register on user site
- algorithm that:
  -  generates meal plans for day/week/month in custom number of meals per chosen period 
  -  calculates user BMI and generates menus based on user's caloric needs
- displays results in web application
